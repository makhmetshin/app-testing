## Содержание работ
В ходе работ было проведено ручное тестирование системы.  

Виды проведенных тестированих:
1) Функциональное тестирование
2) Тестирование UI
3) Smoke тестирование
4) Кроссбраузерное тестирование

### Кроссбраузерный аудит
Успешно

Аспекты, требующие улучшения. 
1)	На странице записи к врачам отсутствует ссылка для возврата на главную страницу, что создает неудобства для пользователей, затрудняющих навигацию по сайту.  
2)	В разделе с картами отсутствует возможность увеличения масштаба, что ограничивает детальное ознакомление с пространственными данными.  
3)	Кроме того, были отмечены некоторые недостатки в правой панели сайта. Иконки представляют собой неинтуитивный интерфейс, что может быть вызвано избыточной нагрузкой сайта различными сервисами.
   
Рекомендуется рассмотреть вышеуказанные предложения для повышения удобства использования и улучшения пользовательского опыта на сайте ЦСМ.
